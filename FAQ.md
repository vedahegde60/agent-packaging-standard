# FAQ

**Q: Is APS another marketplace?**  
A: No. APS is a **format + contract**. Marketplaces can index APS packages.

**Q: Do I have to use Python?**  
A: No. The spec is language-agnostic. We ship a Python example first for convenience.

**Q: Can I self-host agents?**  
A: Yes. APS supports local/VPC execution, and can reference remote endpoints.

**Q: Is monetization required?**  
A: No. Monetization metadata is optional.

**Q: Is this production-ready?**  
A: The spec is in draft. The CLI is a reference to validate the concept.
